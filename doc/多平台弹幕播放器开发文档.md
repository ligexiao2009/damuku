# 多平台本地视频播放器（B站 / 腾讯弹幕支持）开发文档

## 一、项目目标

构建一个本地网页视频播放器，支持：

- 本地视频文件播放（MP4 / MKV 转码）
- B站弹幕加载
- 腾讯视频弹幕加载
- 前端切换弹幕源
- 后端统一弹幕接口
- 弹幕统一格式输出
- 后续支持 ASS / XML / JSON 扩展

---

## 二、整体架构

```txt
前端 HTML/JS 播放器
    ↓
/api/danmaku?source=bili|qq&id=xxx
    ↓
Node.js 后端
    ├── B站弹幕解析器
    ├── 腾讯弹幕解析器
    └── 缓存系统
    ↓
统一JSON弹幕格式
```

## 七、腾讯弹幕支持

- VID


### 时间分段：
- 每30秒一段
- 单位：毫秒

### 示例：

---

### 主路由：

```js
app.get('/api/danmaku', async (req, res) => {
  const { source, id, duration } = req.query;

  let danmaku = [];

  if (source === 'bili') {
    danmaku = await fetchBiliDanmaku(id);
  }

  if (source === 'qq') {
    danmaku = await fetchTencentDanmaku(id, Number(duration));
  }

  res.json(danmaku);
});
```

## 九、腾讯弹幕抓取函数

```js
async function fetchTencentDanmaku(vid, durationMs) {
  const step = 30000;
  let result = [];

  for (let start = 0; start < durationMs; start += step) {
    const end = start + step;

    const url = `https://dm.video.qq.com/barrage/segment/${vid}/t/v1/${start}/${end}`;

    const res = await axios.get(url, {
      headers: {
        Referer: 'https://v.qq.com/',
        Origin: 'https://v.qq.com'
      }
    });

    const parsed = (res.data.barrage_list || []).map(d => ({
      text: d.content,
      time: d.time_offset / 1000,
      color: '#ffffff',
      mode: 'scroll',
      ctime: d.create_time
    }));

    result.push(...parsed);
  }

  return result.sort((a, b) => a.time - b.time);
}
```
